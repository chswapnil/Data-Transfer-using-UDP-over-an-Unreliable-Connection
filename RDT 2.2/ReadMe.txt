Purpose of the Project
To implement an Reliable Data Transfer (RDT) Protocol 2.2 over an unreliable network using User Datagram Protocol (UDP)

Steps required to setup and execute the program:
Step 1: Open the server.py in IDLE
Step 2: Open the Client.py in IDLE
Step 3: Run the server.py
Step 4: Run the client.py
Step 5: Follow the instructions in the command prompt
