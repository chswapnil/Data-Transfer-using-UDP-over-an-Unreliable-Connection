Purpose of the Project
To implement Reliable Data Transfer (RDT) Protocol 3.0 over an unreliable network using User Datagram Protocol (UDP)

Steps required to setup and execute the program:
Step 1: Open the Server.py in IDLE
Step 2: Open the Client.py in IDLE
Step 3: Open the Router.py in IDLE
Step 4: Run the Router.py
Step 5: Select the appropriate condition for transfer
Step 6: Run the Server.py
Step 7: Run the Client.py
Step 8: Follow the instructions in the command prompt
